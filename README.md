# Traccar UI POC

To install dependencies:

```bash
bun install
```

To start a development server (default port 3000):

```bash
bun dev
```

## Traccar POC commands

This app connects to Traccar in realtime via WebSocket. Configure the connection using the in-app **Settings** (enter the WebSocket URL and optional token). These values are persisted in your browser's `localStorage` and will be used automatically.

Start the development server with:

```bash
bun dev
```
